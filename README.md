# サービス概要
目標に向けて継続を支援するサービス
# サービス開発の背景
目標実現のため、多くの人が継続を楽しめるようなサービスを作ってみたかった。またNext.js、CSS in JS、TSのキャッチアップを目的に作成しようと思いました。  
# 使用技術

## 言語・フレームワーク（使用予定）
・Next.js  
・TypeScript  
・Prisma  
・RadixUI  
・Stitches

## インフラ
・Vercel

## 認証・認可  
・Auth0


# 実装機能
・ログイン  
・継続内容投稿機能  
・日々の投稿機能  
・他ユーザーの継続内容に対する応援コメント機能  
・いいね機能（継続内容、質問解答に対して）  
・おすすめ質問機能  
・おすすめ解答機能  
・記事ストック機能（自分の記事、ストックしておきたい記事  
・積読機能
